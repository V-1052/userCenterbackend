# userCenterbackend

> A lightweight user center backend service (authentication, user profile, and related APIs).  

---

## ✨ Features

- User registration / login / logout  
- User profile CRUD  
- Session based auth
- Basic validation & error handling  

---

## 🏗 Tech Stack | 技术栈

- **Language**: Java
- **Build**: Maven
- **Framework**: Spring Boot 3.0.4
- **Database**: MySQL
- **ORM**: MyBatis

---

## 📦 Project Layout | 项目结构

userCenterbackend/
